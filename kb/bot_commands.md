# Bot Commands

Here is the list of available commands for "حواس" (Hawas):

## User Commands
- `/start`: Open the main menu and welcome screen.
- `/status`: Check the systems status (Memory, KB, and Transactions).
- `/kb_list`: List all available knowledge base topics.
- `/kb_search <query>`: Search for information in the knowledge base.

## Admin Commands (Owner Only)
- `/kb_add <title> | <content>`: Add a new document to the knowledge base.
- `/kb_get <id>`: Retrieve the raw content of a doc by ID.
- `/skill_add`: Register a new deterministic skill.
- `/memory_dump`: View stored facts about users.

You can also talk to Hawas in natural Egyptian Arabic to perform transfers or ask questions.
